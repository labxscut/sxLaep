sxLaep

Rapid Enzyme / Non-Enzyme Prediction — an efficient machine learning tool combining multi-physicochemical sequence features with an XGBoost classifier for fast and accurate binary classification of protein sequences.

View on GitHub Get Started API Reference

Overview

Accurate identification of enzymes from protein sequences is crucial for understanding metabolic pathways, drug target discovery, and functional annotation of novel proteins.

  • Fast inference — pre-trained XGBoost classifier optimized for rapid enzyme prediction on large-scale protein datasets
  • Rich feature extraction — combines Pse-AAC, CTD, windowed AAC, and sequence length for comprehensive sequence representation
  • Easy to use — dual Python API and CLI interface; install via pip and start predicting in minutes
  • Bioinformatics ready — built-in FASTA parsing, batch processing, and parallelized feature extraction for production workflows
  • Docker support — pre-configured image with Python 3.11 and all dependencies for isolated execution
  • Built-in evaluation — classification reports, confusion matrices, and accuracy scoring for model assessment

Pipeline

Input FASTA → Feature Extraction → XGBoost Classifier → Enzyme / Non-Enzyme

Installation

sxLaep supports Python 3.7+ on Windows, Linux, and macOS.

From PyPI

pip install sxlaep

From Source

git clone https://github.com/labxscut/sxLaep
cd sxLaep
pip install -e .

Docker

docker pull labxscut/sxlaep:python3.11
docker run --rm -it -v $(pwd):/workspace labxscut/sxlaep:python3.11 bash

Quick Start

Predict enzyme probability for a single protein sequence or a FASTA file in just a few lines of code.

Python API

from sxlaep import sxlaep

predictor = sxlaep()

# Single sequence prediction
pred = predictor.predict("MKVLW...")         # 0 (Non-Enzyme) or 1 (Enzyme)
prob = predictor.predict_proba("MKVLW...")   # float (0.0 ~ 1.0)

# FASTA file prediction
results = predictor.predict_fasta("sequences.fasta")
# {seq_id: {'prediction': int, 'probability': float}}

Command Line

sxlaep --input sequences.fasta --output predictions.json

CLI Usage

# Training
sxlaep train \
  --noenzyme-fasta data/noenzyme.fasta \
  --enzyme-fasta data/enzyme.fasta \
  --outdir results/sxlaep_training

# Prediction
sxlaep predict \
  --model results/sxlaep_training/enzyme_xgb_model.pkl \
  --fasta data/query.fasta \
  --output results/query_predictions.csv

Workflow

End-to-end workflow from raw FASTA files to trained model and predictions.

FASTA Files→ Read & Label→ Feature Extraction→ Train/Test Split→ XGBoost Training→ Evaluate→ Save Model

Training Example

pip install sxlaep

# CLI
sxlaep train \
  --noenzyme-fasta data/noenzyme.fasta \
  --enzyme-fasta data/enzyme.fasta \
  --outdir results/sxlaep_training

# Python API
from sxlaep.training import train_enzyme_classifier

results = train_enzyme_classifier(
    noenzyme_fasta="data/noenzyme.fasta",
    enzyme_fasta="data/enzyme.fasta",
    output_dir="results/sxlaep_training"
)
print(results["accuracy"])  # e.g. 0.92

Prediction Example

# CLI
sxlaep predict \
  --model results/sxlaep_training/enzyme_xgb_model.pkl \
  --fasta data/query.fasta \
  --output results/query_predictions.csv

# Python API
from sxlaep.model import predict_fasta

df = predict_fasta(
    model_path="results/sxlaep_training/enzyme_xgb_model.pkl",
    fasta_path="data/query.fasta",
    output_csv="results/query_predictions.csv"
)
print(df.head())

Output Format

{
  "seq_id_1": { "prediction": 1, "probability": 0.876 },
  "seq_id_2": { "prediction": 0, "probability": 0.234 }
}

# prediction: 0 = Non-Enzyme, 1 = Enzyme
# probability: enzyme probability score (0.0 ~ 1.0)

Notes

  • The pre-trained model (enzyme_xgb_model.pkl) is included in the package.
  • Sequences containing non-standard amino acids are automatically sanitized.
  • Short sequences (< 10 amino acids) may yield less reliable predictions.
  • For large-scale predictions (>10,000 sequences), consider batch processing.
